EN2H Booking Platform API

A RESTful backend application built using NestJS,
PostgreSQL and JWT authentication.

The system allows authenticated users to manage
services and customers to create bookings.

#Tech stack
TypeScript
PostgreSQL
TypeORM
JWT
Swagger



#installation
git clone https://github.com/SajanaWaidyarathna/booking-system.git

cd en2h-booking-api

npm install


#environment variables
DATABASE_HOST=
DATABASE_PORT=
DATABASE_USERNAME=
DATABASE_PASSWORD=
DATABASE_NAME=

JWT_SECRET=


#Api documentation
http://localhost:3000/api
